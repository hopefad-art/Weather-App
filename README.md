## About
This is a weather app that shows current conditions and a 5-day forecast for any city. I built it using HTML for structure, CSS for styling, and JavaScript for the logic. It fetches data from the Open-Meteo API in two steps: first converting a city name into coordinates using the geocoding endpoint, then using those coordinates to get the current weather and forecast. The app uses async/await to handle the API calls and updates the page dynamically with the results, including error handling for when a city isn't found.
